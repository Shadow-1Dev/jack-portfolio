<div>
  <h1>Jack Portfolio</h1>

  <p>
    <strong>About the Project:</strong>
    A personal portfolio website for Jack, a UI Designer and Front-End Developer based in London. It presents his services, skill levels, selected projects, and a contact form — all in a clean, modern layout built with HTML, CSS, and JavaScript.
  </p>

  <p>
    <strong>Key Highlights:</strong>
    Smooth scroll reveal animations, a sticky header on scroll, mobile-friendly navigation, and a fully responsive layout across all screen sizes.
  </p>
</div>

<details open>
  <summary><h2>Project Details</h2></summary>

  <details>
    <summary><h4>What's Inside</h4></summary>
    <ul>
      <li><strong>Hero Section</strong> — Introduces Jack with a photo, title, and call-to-action buttons.</li>
      <li><strong>About Section</strong> — Displays services, skill progress bars, and service cards.</li>
      <li><strong>Projects Section</strong> — Showcases four selected design and development projects.</li>
      <li><strong>Contact Section</strong> — Includes a contact form with name, email, and message fields.</li>
      <li><strong>Header & Footer</strong> — Sticky navigation with logo, links, and social media icons.</li>
    </ul>
  </details>

  <details>
    <summary><h4>Key Features</h4></summary>
    <ul>
      <li><strong>Scroll Reveal Animations</strong> — Elements animate into view as the user scrolls down.</li>
      <li><strong>Sticky Header</strong> — Header fixes to the top with a slide-in animation on scroll.</li>
      <li><strong>Mobile Navigation</strong> — Off-canvas sidebar menu with overlay for small screens.</li>
      <li><strong>Skill Progress Bars</strong> — Animated bars showing proficiency levels for each skill.</li>
      <li><strong>Fully Responsive</strong> — Layout adapts cleanly from 375px mobile to 1400px+ desktop.</li>
      <li><strong>Blob Image Mask</strong> — Hero photo styled with an SVG blob shape using CSS masking.</li>
    </ul>
  </details>

  <details>
    <summary><h4>Technologies Used</h4></summary>
    <ul>
      <li><strong>HTML5</strong> — Semantic markup with accessible landmark roles and attributes.</li>
      <li><strong>CSS3</strong> — Custom properties, flexbox, grid, animations, and media queries.</li>
      <li><strong>JavaScript (ES6)</strong> — Handles navigation toggle, header scroll, and reveal logic.</li>
      <li><strong>Google Fonts</strong> — Manrope font loaded via the Google Fonts API.</li>
      <li><strong>Ionicons</strong> — Icon library used for navigation and social media icons.</li>
    </ul>
  </details>

  <details>
    <summary><h4>Quick Start</h4></summary>
    <ol>
      <li>
        <strong>Clone the repository:</strong>
        <pre><code>git clone https://github.com/nawazdevx/jack-portfolio.git</code></pre>
      </li>

      <li>
        <strong>Open the project:</strong>
        <ul>
          <li>Open <code>index.html</code> directly in your browser</li>
          <li>Or run a local server:</li>
        </ul>
        <pre><code>python -m http.server 3000</code></pre>
        Then visit <code>http://localhost:3000</code>
      </li>

      <li>
        <strong>Start Customizing:</strong>
        <ul>
          <li>Update name, title, and intro text in <code>index.html</code></li>
          <li>Replace the hero photo and project images in <code>assets/images/</code></li>
          <li>Adjust skill percentages and service card content in <code>index.html</code></li>
          <li>Change colors and fonts using CSS variables in <code>assets/css/style.css</code></li>
          <li>Update project cards with your own titles, descriptions, and links in <code>index.html</code></li>
        </ul>
      </li>
    </ol>

  </details>
